# Security Policy — EVIQ AI

We take the security of EVIQ AI, our vehicle integrations, and charging operator connections seriously.

## Supported Versions

We actively support and patch vulnerabilities in the following versions:

| Version | Supported |
| :------ | :-------- |
| v1.0.x  | Yes       |
| < v1.0  | No        |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please do **NOT** report it publicly on GitHub issues.

Instead, report it directly to our security team:

1. Email a detailed report to [security@eviq.ai](mailto:security@eviq.ai) or [founders@eviq.ai](mailto:founders@eviq.ai).
2. Include a description of the vulnerability, steps to reproduce, and possible mitigations.
3. We will acknowledge receipt of your report within 48 hours and work with you to resolve it.

We request that you practice responsible disclosure and give us reasonable time to patch the issue before making it public.
